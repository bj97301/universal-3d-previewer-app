# Report a Bug

Use this page if Universal 3D Previewer is not working correctly.

Before reporting a bug, please try the troubleshooting steps:

[Restart Finder and refresh the Quick Look cache](TROUBLESHOOTING.md)

## What to Include

A good bug report includes:

- Your macOS version
- The file type you tried to preview
- What happened
- What you expected to happen
- Whether restarting Finder or refreshing the Quick Look cache helped

## Submit the Bug Report

When you are ready, open the bug report form:

[Open the bug report form](https://github.com/bj97301/universal-3d-previewer-app/issues/new?template=bug_report.yml)

GitHub may call this an issue, but the form is specifically for bug reports.
