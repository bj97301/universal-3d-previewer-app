# Sample Files

These files are provided for App Review and for users who want quick test files for Universal 3D Previewer.

Download a file, then select it in Finder and press **Space** to open Quick Look. You can also view it as a Finder thumbnail or open it directly in Universal 3D Previewer.

## Files

- [sample_cuboid.step](sample_cuboid.step) - STEP cuboid sample
- [sample_triangle.stl](sample_triangle.stl) - STL mesh sample
- [sample_cube.obj](sample_cube.obj) - OBJ mesh sample
- [sample_cube.ply](sample_cube.ply) - PLY mesh sample
- [sample_triangle.3mf](sample_triangle.3mf) - 3MF mesh sample
- [sample_toolpath.gcode](sample_toolpath.gcode) - G-code toolpath sample

## Troubleshooting

If Finder does not update previews right away, see the troubleshooting guide:
