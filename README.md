# Universal 3D Previewer Help

Universal 3D Previewer is a macOS app that adds Finder previews and thumbnails for common 3D model and toolpath files.

You can use this page for help with the Mac App Store version of the app. You do not need a GitHub account to read these instructions.

## What the App Does

Universal 3D Previewer lets you quickly inspect supported files in Finder before opening a CAD, slicer, or modeling app.

Supported file types include:

- STEP / STP
- STL
- OBJ
- PLY
- 3MF
- G-code
- Alembic / ABC

The app focuses on fast previews. Some complex CAD files may not render with perfect CAD fidelity.

## If Finder Shows Old or Missing Previews

macOS can keep Finder thumbnails and Quick Look previews cached after installing or updating an app.

Start here:

[Restart Finder and refresh the Quick Look cache](TROUBLESHOOTING.md)

That guide walks you through the Terminal commands to restart Finder and clear stale preview caches.

## Report a Bug

If the app is not working correctly, you can open a public bug report here:

[Open a bug report](https://github.com/bj97301/universal-3d-previewer-app/issues/new)

Please include:

- Your macOS version
- The file type you tried to preview
- What happened
- What you expected to happen
- Whether the troubleshooting steps helped

If you do not have a GitHub account, you can still read the help pages here. GitHub requires an account only when creating a new issue.
